# DecodeX 2026 Hackathon – VoltRide Expansion Strategy

This project was developed during the DecodeX 2026 Hackathon.

## Objective
Analyze the VoltRide dataset and recommend a strategy for expansion after a regime shift in the EV mobility market.

## Key Tasks
- Data cleaning and exploratory analysis
- Identifying demand and supply patterns
- Evaluating expansion strategies
- Building decision models

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel

## Outcome
Proposed a strategic expansion plan considering demand changes, infrastructure constraints, and operational risks.
